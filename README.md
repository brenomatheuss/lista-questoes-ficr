# 📘 Lista de Exercícios em C – 

Este repositório contém uma coleção de **13 algoritmos em linguagem C**, desenvolvidos com o objetivo de praticar **comandos sequenciais**, entrada de dados, operações matemáticas e lógica básica de programação.

---

## 🧠 Conteúdos abordados

* Entrada e saída de dados (`scanf` e `printf`)
* Operações matemáticas básicas
* Estruturas sequenciais
* Conversões de unidades
* Cálculos geométricos
* Controle de erros (ex: divisão por zero)
* Manipulação de variáveis

---

## ⚙️ Como executar os programas

1. Instale um compilador C (ex: GCC, OnlineGDB, Falcon C++, etc.)
2. Salve o código em um arquivo `.c`

---

## 📌 Observações

* Todos os exercícios utilizam apenas **estrutura sequencial**
* Ideal para iniciantes em programação
* Pode ser executado em qualquer compilador C padrão
